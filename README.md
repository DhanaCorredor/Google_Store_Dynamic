Este proyecto es un desarrollo individual realizado por **Dhana Corredor** en el marco del bootcamp de **Factoría F5**. Se centra en la creación de una interfaz dinámica que replica la experiencia de usuario de la Google Store, priorizando la lógica de programación y la manipulación del DOM.

---

## 🚀 Enfoque en JavaScript
El núcleo de este repositorio es el desarrollo de funcionalidades interactivas mediante JS "Vanilla":

* **Gestión del Carrito:** Implementación de un sistema de compras dinámico.
* **Persistencia de Datos:** Uso de `localStorage` para mantener los productos tras recargar la página.
* **Galería Dinámica:** Intercambio de imágenes de producto mediante eventos de clic.
* **Validación de Formularios:** Control de selección de variantes (colores y cantidades) antes de añadir al carrito.
