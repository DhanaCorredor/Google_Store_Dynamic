## 📖 Descripción

Tienda online interactiva con carrito persistente, galería de imágenes dinámica y validaciones robustas. Implementado en JavaScript vanilla.

## ✨ Características

- 🛒 Carrito persistente con localStorage
- 🎨 Galería dinámica con vista previa de colores
- ✅ Validaciones robustas de datos
- 📱 Diseño responsivo (mobile-first)
- 📦 Soporte multi-página (Earbuds, Watches)
- 💾 Sincronización automática de carrito

## 🚀 Instalación

```bash
git clone https://github.com/DhanaCorredor/Google_Store_Dynamic.git
cd Google_Store_Dynamic
python -m http.server 8000
```

Abre `http://localhost:8000` en tu navegador.
